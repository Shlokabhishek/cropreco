üìñ GLOSSARY OF TERMS

Term Definition
AGMARKNET Agricultural Marketing Information Network - Government of India's portal for commodity prices
DanfoJS JavaScript/TypeScript library for data manipulation (similar to pandas in Python)
Min-Max Normalization Scaling technique that transforms values to [0, 1] range
MSP Minimum Support Price - Government-declared minimum price for crops
Quintal Unit of weight = 100 kg (commonly used for agricultural commodities in India)
ROI Return on Investment - Ratio of profit to investment cost
Median Middle value in sorted dataset (robust against outliers)
Intercropping Growing two or more crops simultaneously on same field
Kharif Monsoon cropping season (June-October)
Rabi Winter cropping season (October-March)
JWT JSON Web Token - Secure authentication token standard
API Application Programming Interface - System for external data access
